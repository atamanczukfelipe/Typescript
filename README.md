# Typescript
Typescrip Full

Init and create project package.json
```javascript 
npm init
```

Create ts config.json
```javascript 
npx tsc --init
```

Install Ts Node Dev
```javascript 
npm i ts-node-dev -D
```

Execute and Trasnpile the ts file
```javascript 
npx ts-node-dev app.ts
```

```javascript 
npx ts-node-dev [file name]
```
